## 歡迎

- NapCatQQ 是現代化的基於 NTQQ 的 Bot 協議端實現

## 特性

- **易於使用**
  - 初學者也能輕鬆上手.
- **快速高效**
  - 在低記憶體系統長時間穩定運行.
- **豐富的 API 介面**
  - 完整實現了大部分標準介面.
- **穩定可靠**
  - 持續穩定的開發與維護.

## 快速開始

可前往 [Release](https://github.com/NapNeko/NapCatQQ/releases/) 頁面下載最新版本

**首次使用**請務必查看以下文件中的使用教學
