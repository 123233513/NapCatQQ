## 相关提示
* 由于该项目特殊性，NapCat/Core 部分没有开源打算，可能会开放CoreApi以便开发与适配协议实现。

* 请不要在无关地方 宣传NapCat，违法使用NapCat。

* NapCat 不会收集用户隐私信息，但是未来可能会为了更好的利于 NapCat 的优化会收集一些设备信息，如 cpu 架构，系统版本等
